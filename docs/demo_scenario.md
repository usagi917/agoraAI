# Agent AI デモシナリオ

## デモ1: ビジネス分析（Single モード）

### 入力
- ファイル: `sample_inputs/business_case/market_entry.md`（EVバッテリー市場参入）
- テンプレート: ビジネス分析
- プロファイル: Preview（高速）

### 見せるポイント
1. ドキュメントアップロード → 世界モデル自動構築
2. 3Dフォースグラフでエンティティ関係を可視化
3. エージェントがラウンドごとに意思決定を繰り返す
4. レポート生成（11セクション構造化出力）
5. フォローアップ質問でレポートエージェントと対話

## デモ2: Swarm 分析（Swarm モード）

### 入力
- プロンプト: 「2030年のAI規制はどうなるか？楽観・中立・悲観の3シナリオで分析せよ」
- テンプレート: シナリオ探索
- プロファイル: Standard

### 見せるポイント
1. 複数Colonyが異なる視点（楽観/中立/悲観/アドバーサリアル）で並列実行
2. 各Colonyの進捗がリアルタイムで表示
3. シナリオ比較 — 確率分布、信頼区間
4. 合意ヒートマップ — Colony間の意見の一致度
5. 多様性スコア、エントロピーの統計

## デモ3: PM Board モード

### 入力
- プロンプト: 「B2B SaaS型の建設プロジェクト管理ツールの事業化を検討する」
- モード: PM Board

### 見せるポイント
1. 4人のPMペルソナ（戦略PM、ディスカバリーPM、実行PM、チーフPM）が並列分析
2. 11セクション構造化出力:
   - 核心質問、前提条件（確信度付き）、不確実性、リスク
   - 勝利仮説（IF-THEN-TO ACHIEVE形式）
   - 顧客検証計画、市場/競合ビュー、GTM仮説
   - MVPスコープ、30/60/90日計画
   - 今すぐやるべき5アクション
3. PM間の矛盾検出と解決案
4. 総合確信度スコア

## デモ4: 認知アーキテクチャの深さ（Cognitive Advanced モード）

### 入力
- ファイル: `sample_inputs/policy_case/carbon_tax.md`
- テンプレート: 政策シミュレーション
- プロファイル: Standard

### 見せるポイント
1. **BDI状態**: 各エージェントの信念・欲求・意図がラウンドごとに変化
2. **記憶ストリーム**: エピソード記憶 → 意味記憶 → 省察の流れ
3. **Theory of Mind**: エージェントが互いの目標を推論
4. **社会ネットワーク**: 同盟・対立関係の動的変化
5. **ナレッジグラフ**: 文書から自動構築された知識構造
