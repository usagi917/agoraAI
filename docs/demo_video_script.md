# Agent AI デモ動画スクリプト（2-3分）

## 構成

### 1. フック（0:00 - 0:10）

**ナレーション:**
「ChatGPTに聞いても、いつも同じ視点。もし20人のAIエージェントが、それぞれ独自の認知モデルで議論したら？」

**映像:** 3Dフォースグラフが回転、ノードが光り、エッジが脈動

---

### 2. コンセプト紹介（0:10 - 0:30）

**ナレーション:**
「Agent AI は BDI認知アーキテクチャを搭載。各エージェントは独自の信念、欲求、意図を持ち、Theory of Mind で互いの思考を推論しながら議論します。」

**映像:**
- アーキテクチャ図のアニメーション
- BDI状態が変化する様子（認知状態タブ）
- ToMマップでエージェント間の推論が表示される

---

### 3. デモ: ドキュメント分析（0:30 - 1:30）

**ナレーション:**
「使い方はシンプル。ドキュメントをアップロードして、分析モードを選ぶだけ。」

**映像:**
1. LaunchPad画面でファイルアップロード（3秒）
2. テンプレート選択 → 実行ボタン押下（3秒）
3. シミュレーション進行中の画面:
   - 3Dグラフが成長（10秒、早回し）
   - エージェントの行動ログが流れる
4. 結果画面:
   - レポートタブ: 構造化レポートをスクロール（10秒）
   - 3Dグラフ: テンポラル再生で時間経過を表示（5秒）
   - シナリオ比較: 確率分布チャート（5秒）
   - 合意ヒートマップ: Colony間の合意度（3秒）

---

### 4. デモ: PM Board（1:30 - 2:10）

**ナレーション:**
「PM Board モードでは、5人のPMペルソナが同じ入力を並列分析。戦略、ディスカバリー、実行の各視点から構造化された意思決定ドラフトを生成します。」

**映像:**
1. PM Board モード選択（2秒）
2. プロンプト入力:「新規SaaSの事業化検討」（3秒）
3. PM分析中の進捗表示（5秒、早回し）
4. 結果画面:
   - 核心質問 → 前提条件（確信度バー）（5秒）
   - 勝利仮説（IF-THEN-TO ACHIEVE）（5秒）
   - 30/60/90日計画のタイムライングリッド（5秒）
   - 今すぐやるべき5アクション（5秒）

---

### 5. 比較: ChatGPT vs Agent AI（2:10 - 2:30）

**映像:** 画面2分割

| ChatGPT | Agent AI |
|---------|----------|
| 単一視点の回答 | 複数エージェントの構造化議論 |
| テキストのみ | 3Dグラフ + 構造化レポート |
| 「〜が考えられます」 | 確信度付き仮説 + 検証計画 |
| 1回きりの出力 | ラウンドごとの進化 |

## 制作メモ

### 必要なツール
- 画面収録: OBS Studio / ScreenFlow
- 編集: DaVinci Resolve (無料版で十分)
- BGM: 著作権フリーのテック系BGM
- ナレーション: 合成音声（VOICEVOX / ElevenLabs）

### 画面設定
- 解像度: 1920x1080
- ブラウザ: Chrome（フルスクリーン、ブックマークバー非表示）
- フォントサイズ: 少し大きめに調整
